# **App Name**: HandyTask

## Core Features:

- Task Dashboard: Dashboard providing an overview of all handyman tasks.
- Ticket Submission: Allows clients to submit detailed hardware job requests including address and problem description.
- Task Assignment: Functionality to assign submitted tasks to available handymen.
- Status Tracking: Progress indicators for each task (e.g., To Do, In Progress, Completed).
- AI-Powered Troubleshooting: Suggesting possible root causes of hardware malfunctions or optimal solutions based on the ticket description using an AI tool.
- Integrated Communication: Integrated chat feature for communication between clients and assigned handymen.
- Update Job status: Enable handymen to update the job status

## Style Guidelines:

- Primary color: A calm, professional blue (#468FAF) to inspire trust.
- Background color: Light gray (#F0F4F7) for a clean and modern feel.
- Accent color: Soft teal (#4F9A8A) for interactive elements and call-to-actions.
- Clear, sans-serif font for easy readability across all devices.
- Consistent use of line icons for task statuses and navigation.
- Clean, card-based layout to organize tasks efficiently.
- Subtle transitions and animations for a smooth user experience.
